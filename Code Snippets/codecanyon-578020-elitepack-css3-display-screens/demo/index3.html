Monitor

Monitors


Elitepack CSS3 monitors are a nice modern way of showcasing your application, website or portfolio gallery using pure CSS3 powered models. The monitor pack features some nice graphical elements to showcase your application, website or portfolio on a desktop monitor.

Let's start off with a simple monitor


Here is a breakdown of the classes used to create the laptop:

  • 'monitor' class defines the main frame of the monitor container
  • 'screen-m' class defines the display screen element
  • 'base-m' class defines the connector between the monitor and the stand
  • 'stand-m' class defines the bottom most section of the stand

Optional Classes

  • 'glare-m' class adds a glare light effect to the monitor display screen
  • 'glow-m' class adds a glow light effect to the monitor display screen
  • 'top-base-shadow' class defines a shadow from a top light onto the monitor
  • 'black-m' class defines a monitor without an extended white space slightly below the display screen area

  • View HTML Code

    <div class="monitor" >
<div class="screen-m"></div>  
<div class="base-m"></div>  
<div class="stand-m"></div> 
</div>

Adding a background image


To add an image to the monitor, simply add the <img> tag within the div of the screen-m class. By default, the dimensions of the image has been set to 422px by 225px. You'll notice if you use a bigger image it will be resized to fit to these dimensions. However, this can be changed in the css stylesheet

  • View HTML Code

    <div class="monitor">
	<div class="glare-m"></div>
	<div class="screen-m">
		<img src="images/monitor.png" alt="monitor" />  
	</div>
	<div class="base-m"></div>
	<div class="stand-m"></div>
</div>

Monitor with a youtube or vimeo video


To add a hosted video simply add the <iframe> tag within the div of the screen-m class.

  • View HTML Code

    <div class="monitor">
	<div class="glare-m"></div>
	<div class="screen-m">
		<iframe width="422" height="225" frameborder="0" src="http://player.vimeo.com/video/7449107?title=0&amp;byline=0&amp;portrait=0&amp;color=963695"></iframe>
	</div>
	<div class="base-m"></div>
	<div class="stand-m"></div>
</div>

Adding a shadow to a monitor


Shadows are a nice way to even further add '3D effects' to your monitor. you can add a shadow by adding a div with the top-base-shadow class between the base-m class and the stand-m class

  • View HTML Code

    <div class="monitor">
	<div class="glare-m"></div>
	<div class="screen-m">
		<img src="images/monitor.png" alt="monitor" />  
	</div>
	<div class="base-m"></div>
	<div class="top-base-shadow"></div>
	<div class="stand-m"></div>
</div>

Black Monitor


To create a black monitor (black in the sense that there is no extended white portion beneath the main display screen area - which is black), add the black class within the monitor div

  • View HTML Code

    <div class="monitor black">
	<div class="glare-m"></div>
	<div class="screen-m">
		<img src="images/monitor_h.png" alt="desktop" />  
	</div>
	<div class="base-m"></div>
    <div class="top-base-shadow"></div>
	<div class="stand-m"></div>
</div>

Monitors stacked together

To stack monitors together, add the left-stack class within the monitor div of the left most monitor, main-stack class to the center monitor or right-stack class to the monitor div of the right most monitor. In the example above I have used left-stack and main stack. You can optionally use the mini together with the stack class to reduce the size of the monitors

  • View HTML Code

    <div class="monitor mini left-stack">
	<div class="glare-m"></div>
	<div class="screen-m">
		<img class="vis65" src="images/monitor.png" alt="monitor" />  
	</div>
	<div class="base-m"></div>
	<div class="top-base-shadow"></div>
	<div class="stand-m"></div>
</div>							
<div class="monitor mini main-stack">
	<div class="glare-m"></div>
	<div class="screen-m">
		<img src="images/monitor.png" alt="monitor" />
	</div>
	<div class="base-m"></div>
	<div class="top-base-shadow"></div>
	<div class="stand-m"></div>
</div>
