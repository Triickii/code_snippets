1. What is Elitepack CSS3?

Elitepack CSS3 is a collection of web elements that are designed with pure CSS3 technology meaning that no images have been used except the small icons or background images. This particular pack is the CSS3 display screens which are a nice modern way of showcasing your application, website or portfolio gallery using CSS3 powered display 'vectors'

By using CSS3 and a few lines of HTML code to create display screens, it greatly improves the load time of your site, while giving you the flexibility to change dimensions, orientation, color, functionality and practically anything without opening your favorite graphic software.

2. CSS File Structure

Once you download the zipped file, extract it to a local folder or on your server via FTP. You will find 2 folders within: demo and css:

  • demo - This is a working HTML version showcasing the css3 tags in use
  • css - the css3 display screens pack. To use it in your website, copy the entire css folder and paste it in the root folder of your site then reference the css stylesheet appropriately

3. Including the CSS in your HTML/PHP header

NOTE: This section assumes you have the basic knowledge in HTML/PHP and how to apply CSS stylesheets in your header.

To add the css stylesheets into your website (whether HTML, PHP or any other language that utilizes stylesheets) simply add the following code in the head section of your HTML File.

  • View the code

    <link rel="stylesheet" href="css/phone.css" type="text/css" media="screen" />

4. Implementing the CSS3 display screens

Once you have included your preferred css stylesheet in your header section, you are ready to roll. To create a phone using CSS3, insert a div tag (denoted as <div class="tablet">) and add a class with the name 'tablet'. Within the class section, add three other divs and insert the classes glare-t, screen-t, addons-t, and button-t in that order.

  • code to create a tablet:

    <div class="tablet white">
<div class="glare-t"></div> 
<div class="screen-t"></div>
<div class="addons-t"></div>
<div class="button-t"></div>  
</div>

5. Adding media content to your display

Once you create your display screens, you can add any kind of media within the 'screen-?' div where ? stands for the letter denoting the different display screens. e.g. screen-p for phone.css. You can use images, slideshows, iframes to show hosted videos or webpages, HTML5 video and pratically any other media.

6. Increasing size and changing dimensions

There are two ways that you can increase or decrease the size of a display screen:

  • By using the transform element of CSS3: you do so by adding a scale vector to the outermost div e.g. tablet or laptop to increase or decrease the size of the display screen. The default size is 1.
  • By adjusting the width and height uniformly until the desired size is attained. Use a web css tool like firebug to help you see the actual effect of the adjustments
