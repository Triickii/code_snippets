Laptop

Laptops


Elitepack CSS3 laptops are a nice modern way of showcasing your application, website or portfolio gallery using pure CSS3 powered models. The laptop pack features some nice graphical elements to showcase your application, website or portfolio on a notebook.

A basic laptop

Here is a breakdown of the classes used to create the laptop:

  • 'laptop' class defines the main frame of the laptop container
  • 'screen-l' class defines the display screen element
  • 'base-l' class defines the bottom section that contains the keyboard
  • 'addons-l' class defines the base screw holes and camera.
  • 'button-l' class defines the button used to open the laptop lid and the front slot

optional classes

  • 'glare-l or glow-l' class adds a light effect to the laptop display screen
  • 'slim' class defines a ultra slim laptop
  • 'main-stack, left-stack and right-stack' classes can be used to stack couple of laptops together
  • 'camera-on' class within the 'addons-l' class will turn the tiny green camera light on

  • View HTML Code

    <div class="laptop" >
	<div class="screen-l"></div>  
	<div class="base-l"></div>  
	<div class="addons-l"></div> 
	<div class="button-l"></div> 
</div>

Laptop with a background image

To add an image simply add the <img> tag within the div of the screen-l class. By default, the dimensions of the image has been set to 365px by 195px. So you'll notice if you use a bigger image it will be resized to fit to these dimensions. However, this can be changed in the css stylesheet

  • View HTML Code

    <div class="laptop">
	<div class="glare-l"></div>
	<div class="screen-l">
		<img src="images/laptop.png" alt="laptop" />  
	</div>
	<div class="base-l"></div>
	<div class="addons-l "></div>
	<div class="button-l"></div>
</div>

Laptop with a youtube or vimeo video

To add a hosted video simply add the <iframe> tag within the div of the screen-l class.

    <div class="laptop">

	<div class="screen-l">
		<iframe width="365" height="195" frameborder="0" src="http://player.vimeo.com/video/7449107?title=0&amp;byline=0&amp;portrait=0&amp;color=963695"></iframe>
	</div>
	<div class="base-l"></div>
	<div class="addons-l "></div>
	<div class="button-l"></div>
</div>

Laptop with a self hosted video using HTML5 semantics

HTML5 Video Player by VideoJS

To add a hosted video simply add the <video> tag within the div of the screen-l class.

Many thanks to videojs for a profound piece of code on how to implement HTML5 video tag with fallback to flash. Be sure to reference all the dimensions and fall back images correctly. Note also for the flash fallback, you've got to specify absolute paths to the video and image placeholder for it to work correctly in older browsers.

  • View HTML Code

    <div class="laptop">
<div class="glare-l"></div>
<div class="screen-l">
<div class="video-container">
<!-- Begin VideoJS -->
<div class="video-js-box">
<!-- Using the Video for Everybody Embed Code http://camendesign.com/code/video_for_everybody -->
<video class="video-js" width="365" height="195" controls preload poster="./video/envato_l.png">
<source src="./video/envato_sting.mp4" type='video/mp4; codecs="avc1.42E01E, mp4a.40.2"' />
<source src="./video/envato_sting.webm" type='video/webm; codecs="vp8, vorbis"' />
<source src="./video/envato_sting.ogv" type='video/ogg; codecs="theora, vorbis"' />
<!-- Flash Fallback. Use any flash video player here. Make sure to keep the vjs-flash-fallback class. -->
<object class="vjs-flash-fallback" width="365" height="195" type="application/x-shockwave-flash"
data="http://releases.flowplayer.org/swf/flowplayer-3.2.1.swf">
<param name="movie" value="http://releases.flowplayer.org/swf/flowplayer-3.2.1.swf" />
<param name="allowfullscreen" value="true" />
<param name="flashvars" value='config={"playlist":["http://demo.arrowthemes.com/video/envato_l.png", {"url": "http://demo.arrowthemes.com/video/envato_sting.mp4","autoPlay":false,"autoBuffering":true}]}' />
<!-- Image Fallback. Typically the same as the poster image. -->
<img src="./video/envato_p_h.png" width="365" height="195" alt="Poster Image"
title="No video playback capabilities." />
</object>
</video>
<!-- Download links provided for devices that can't play video in the browser. -->
<p class="vjs-no-video"><strong>Download Video:</strong>
<a href="./video/envato_sting.mp4">MP4</a>,
<a href="./video/envato_sting.webm">WebM</a>,
<a href="./video/envato_sting.ogv">Ogg</a><br>
<!-- Support VideoJS by keeping this link. -->
<a href="http://videojs.com">HTML5 Video Player</a> by VideoJS
</p>
</div>
<!-- End VideoJS -->

</div>

Different Laptops stacked together

For this set of stacked laptops, I have used a little 'depth' trick by adding an opacity class to the images on the left and right stacked laptops. Notice how the top tablet has a more sharper image than the left and right laptop. You do so by adding, for example, a vis65 class to the images of the left and right stacked laptops to make its opacity be at 0.65

  • View HTML Code

    <div class="laptop mini left-stack">
	<div class="glow-l"></div>
	<div class="screen-l">
		<img class="vis65" src="images/laptop.png" alt="laptop" />
